# mini-redis
C++ in-memory data storage

## info:

- default port: `4242`


Supplied commands:
- `SET [key] [value]`
- `GET [key]`
- `DEL [key]`
- `EXISTS [key]`
- `EXPIRES [key] [ttl]`

note: commands can be wrote in `aNY caSe`
